Python Modules for GLM Neuron Encoding Algorithms

View on GitHub Download .zip Download .tar.gz

This repository contains functions to implement and test the performance of GLM encoding algorithms. It is being tested in Python 3.4 on MAC OS Maverick.

code:

filters.py

  • contains methods for creating filters using cosine bump functions

PPclasses.py

  • contains a class for parameter estimation in non-homogeneous Poisson processes
